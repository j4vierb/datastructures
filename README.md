# Estructuras de datos y algoritmos

En este repositorio se encontraran los temas de estructuras de datos y algoritmos que voy estudiando semana a semana. Esto con el fin de poder desarrollar de manera más sencilla problemas de algoritmia.

Aquí mismo se encontrarán las distintas implementaciones en Java de lo anteriormente mencionado. Asi mismo las notas referentes a cada uno.

Principalmente haré uso de la página de [Geeks for Geeks](https://www.geeksforgeeks.org/), especificamente la sección de [estructuras de datos](https://www.geeksforgeeks.org/data-structures).

## Semana 1 - (12/feb - 18/feb)

La primera estructura de datos serán los [Heaps](https://www.geeksforgeeks.org/heap-data-structure/). Por lo pronto será entender que es un **Heap**, debido a que solo faltan 2 días para que se acabe la semana.
